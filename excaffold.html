Excaffold

>

Esta es una herramienta que facilita el desarrollo en tecnologías como Node JS y el uso de bases de datos relacionales como MYSQL, Postgre SQL y bases de datos no SQL como Mongo DB. Permite con una única línea de comando en consola crear el modelo , la vista y el controlador de una determinada entidad, es decir, que si se desea crear en un sitio web la entidad usuario, su nombre, apellido y mail, se crea de forma automática un archivo que contiene el modelo de dicha entidad, otro que posee el controlador capaz de conectarse a la base de datos y realizar acciones como; crear, modificar, eliminar y leer registros asociados a esta. Junto con esto se crean las vistas, es decir, archivos de representación HTML para enlistar los registros de la tabla de la base de datos asociada a la entidad, y una vista para crear, actualizar y eliminar los registros.
